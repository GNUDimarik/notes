1. Setting Up New GIT repos localy and pushing to GIT HUB
BEGIN 1.

1.2: git init                                        //this will setup git
1.3: <make changes... etc>
1.4: <add and/or commit changes... etc>
1.5: git remote add origin <url to the server repo>  //set remote origin's repo pointer
1.6: git push origin master                          //this will push to the origin's master branch
END 1.
